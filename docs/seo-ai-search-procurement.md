# Arbor SEO + AI Search Procurement Brief

Date: 2026-06-17

## Goal

Give Arbor a serious search foundation for Israel and English expansion:

- Technical SEO and AI crawler access checks.
- Keyword clustering and topic mapping.
- Content briefs for parent-facing and professional-facing searches.
- AI search visibility monitoring across answer engines.
- Schema markup, metadata, hreflang, and sitemap validation.

## Recommendation

Buy a lean stack first. Do not overbuy enterprise SEO software before the custom
domain, Search Console property, and first content clusters are live.

### Purchase Now

1. **Screaming Frog SEO Spider - paid license**
   - Purpose: technical SEO crawler, robots/indexation checks, metadata audit,
     broken links, redirects, structured data, and JavaScript rendering.
   - Why: best value for deep technical auditing. Current public reviews list
     paid pricing around `$279/user/year`.
   - Use for Arbor: weekly crawl of the custom domain, sitemap verification,
     hreflang validation, schema validation, and AI crawler allow/block checks.

2. **Semrush One Starter or SE Ranking Core + AI Search add-on**
   - Purpose: keyword research, competitor research, rank tracking, topic
     clustering, content planning, and AI visibility monitoring.
   - Default choice: **Semrush One Starter** if the budget supports it, because
     it combines classic SEO with AI search visibility in one workflow.
   - Leaner choice: **SE Ranking Core + AI Search add-on** if the priority is
     lower monthly cost with strong SEO/GEO coverage.

### Do Not Purchase Yet

- Enterprise AEO platforms such as Profound unless Arbor has a large content
  footprint, brand search demand, and budget for enterprise contracts.
- Separate schema SaaS tools. Arbor can generate JSON-LD directly in the static
  marketing pages and validate with crawler tooling.
- Expensive content optimization suites before the topic map is proven.

## Capability Mapping

| Need | Buy / Build | Recommended Tool |
|---|---|---|
| Technical SEO & AI crawler audit | Buy | Screaming Frog SEO Spider |
| Keyword clustering & topic mapping | Buy + internal planning | Semrush One or SE Ranking |
| B2B / niche content briefs | Build internally from keyword clusters | Arbor content briefs in `/docs` |
| AI search visibility / AEO | Buy | Semrush One, SE Ranking AI Search, or later Profound |
| Schema markup & metadata generator | Build | Static JSON-LD templates + crawler validation |

## Implementation Plan

1. Purchase and connect Search Console after the Israeli domain is live.
2. Buy Screaming Frog paid license and run the first full crawl.
3. Start Semrush One Starter trial or SE Ranking Core + AI Search add-on.
4. Import the custom domain and sitemap.
5. Track the Hebrew and English keyword clusters above.
6. Export keyword/topic data into Arbor content briefs.
7. Add JSON-LD templates to every marketing page:
   - `Organization`
   - `SoftwareApplication`
   - `FAQPage` where relevant
   - `BreadcrumbList` for guide pages
8. Run weekly technical crawl and monthly AI visibility snapshot.

## Purchase Decision

Recommended first purchase order:

1. Screaming Frog SEO Spider paid license.
2. Semrush One Starter monthly trial.
3. Re-evaluate after 30 days.

If monthly budget is tight:

1. Screaming Frog SEO Spider paid license.
2. SE Ranking Core + AI Search add-on.
